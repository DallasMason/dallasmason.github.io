Project #3: Instructions

Project Description

For this assignment, I was tasked with creating a detailed set of written instructions for a specific audience. I was given the choice to create the instructions on any topic of my choice. The purpose was to demonstrate an understanding of how to go about dividing a complicated task into smaller tasks and use clear, unambiguous language and meaningful illustrations

Skills Learned:

This project strengthened my C programming skills with arrays, strings, and nested loops. I gained experience implementing a classic encryption algorithm, managing input validation, and handling multiple messages. It also helped me develop algorithmic thinking for secure data processing and understand practical applications of cryptography.
