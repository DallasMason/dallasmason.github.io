Project Description

Implemented the Hill cipher encoder/decoder using matrix arithmetic in C++. The program reads a key matrix and plaintext file and outputs ciphertext formatted to assignment specs.

Skills Learned

  • Linear algebra applied to cryptography
  • File I/O and command-line argument parsing in C++
  • Careful handling of padding and modular arithmetic
