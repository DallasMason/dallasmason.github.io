Railfence Encryption

This C program implements the Railfence Cipher encryption method. It reads a user-specified number of messages, then encrypts each using a zigzag pattern on a grid of N rows, applying a starting row shift for added complexity.

Developed for CIS3362 at the University of Central Florida.

Features:

  • Encrypts messages using the Railfence Cipher with user-defined rows and starting row.
  • Supports multiple messages in a single run.
  • Handles newline characters and ensures proper null termination of output.
  • Implements a flexible row-mapping system for added encryption variation.

Usage:

Input the number of messages, followed by the number of rows, starting row, and message text. The program outputs the encrypted message for each input.

Skills Learned:

  • C programming (arrays, strings, and nested loops)
  • Understanding and implementing the Railfence Cipher algorithm
  • Input validation and handling multiple messages
  • Memory management and string manipulation in C
  • Algorithmic thinking for encryption methods
