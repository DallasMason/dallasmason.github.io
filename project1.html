Project Description

This Java application simulates a bank with multiple accounts and synchronized threads to simulate concurrent transactions. It demonstrates proper use of locks, thread coordination, and testing for race conditions.

Skills Learned

  • Concurrent programming in Java (Locks, synchronized blocks)
  • Designing thread-safe data structures
  • Unit testing and debugging concurrency issues
